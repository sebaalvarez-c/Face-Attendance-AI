# Face Attendance AI 

Sistema inteligente de control de asistencia desarrollado mediante Inteligencia Artificial, utilizando reconocimiento facial, análisis de emociones, reconocimiento de voz y geolocalización GPS.

El sistema permite registrar usuarios, identificar personas automáticamente mediante la cámara y almacenar asistencias en una base de datos PostgreSQL.

---

# Características principales

* Reconocimiento facial mediante Deep Learning  
* Generación de embeddings faciales  
* Registro automático de asistencia  
* Análisis de emociones del usuario  
* Reconocimiento de voz como validación adicional  
* Validación mediante ubicación GPS  
* Dashboard para visualizar registros  
* Prevención de asistencias duplicadas
